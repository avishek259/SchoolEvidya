### How do I get set up? ###

Checkout the project in your desktop in some folder 
Configure maven above version 3.0
Move to that Folder 
Run a Maven clean 
Run a Maven install
Move the generated war file in the Local JBoss
Accessible over the http://localhost:8080/schoolapp
